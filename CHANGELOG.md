# 更新日志

## 0.4.6

### 兼容性

- 确认兼容 AstrBot `4.26.4`，插件 API、设置页桥接、WebUI 注入路径、统计页增强和透明化样式锚点均保持可用。
- README 开头补充当前版本已确认兼容 AstrBot `4.26.4` 的说明。

## 0.4.5

### 修复

- 修复横竖屏方向切换时浏览器可能把背景图层更新合并到同一帧，导致叠化看起来像硬切的问题。
- 方向切换现在会等待目标壁纸加载和解码后，再用双层背景触发更明显的 opacity 过渡。
- 保留 0.4.4 的方向竞态保护，快速连续旋转时旧请求不会覆盖新方向背景。

## 0.4.4

### 新增

- 新增横屏/竖屏两套背景图库，可分别上传和选择电脑横屏壁纸、手机竖屏壁纸。
- Dashboard 会按当前视口方向选择对应壁纸，并在旋转或窗口方向变化时以叠化方式切换。
- 打开或刷新时随机背景现在会优先随机当前方向图库，避免横屏和竖屏壁纸混用。

### 兼容性

- 保留旧版 `background_image` 和 `background_images` 配置作为兜底，旧用户升级后原背景仍可继续显示。
- 某个方向没有壁纸时，会自动回退到旧背景或另一方向壁纸，避免空白背景。
- 旧版单图库默认归入横屏图库；删除图片会同步移除横屏、竖屏和旧图库里的同名引用，避免残留失效配置。

## 0.4.3

### 新增

- 新增“系统统计模型 Token 明细”开关，默认关闭，可在插件设置页“高级”标签中启用。
- 开启后会在 AstrBot 系统统计页追加模型 Token 明细面板，展示总计以及每个模型的输入、输出、缓存命中和缓存命中率。
- 新增只读插件接口 `/astrbot_plugin_palette/token-stats`，聚合 AstrBot 已有 `ProviderStat` 记录，不改变模型调用和原生统计接口。

### 兼容性

- 未启用开关时不会向系统统计页插入额外面板。
- 当前环境不支持 `ProviderStat` 明细时，接口会返回不可用状态，避免影响 Dashboard 正常使用。

## 0.4.2

### 修复

- 修复 AstrBot 桌面端或使用内置 WebUI 时，插件只查找 `data/dist/index.html` 导致无法注入背景脚本的问题。
- 注入器现在会跟随 AstrBot 的 WebUI 目录优先级：自定义 `--webui-dir`、兼容的 `data/dist`、内置 `astrbot/dashboard/dist`。
- 内置 WebUI 不可写时，会复制一份到 `data/dist` 并在状态页提示需要重启 AstrBot 后生效。

### 优化

- 设置页状态卡新增注入目标和重启提示，方便桌面端用户判断当前补丁是否已经生效。
- Dashboard 入口备份文件名加入目标路径摘要，避免 `data/dist` 和内置 WebUI 的备份互相覆盖。

## 0.4.1

### 优化

- 图库缩略图改为后端生成并缓存的 `320px` 小图，不再把原图 base64 当作缩略图加载。
- 设置页右侧效果预览和各标签预览改用同一份压缩缩略图，减少云端加载压力。
- 上传新背景图时会预生成缩略图，旧图库图片首次访问缩略图时会自动懒生成。
- 缩略图优先保存为 WebP，当前环境不支持 WebP 保存时自动回退 JPEG。
- 缩略图生成会修正 EXIF 方向，透明图片在 JPEG 回退时会合成浅色底，避免黑底。

### 修复

- 改善云端部署和多张 4K 壁纸场景下，插件设置页图库加载明显偏慢的问题。
- 删除背景图片时同步清理对应缩略图缓存，避免运行时数据残留。

## 0.4.0

### 新增

- 插件设置页重构为分标签布局，拆分为图库、外观、可读性、主题和高级配置。
- 设置页新增 Apple-like Liquid Glass 视觉系统，使用透明中心、边缘高光和轻量折射层替代厚重卡片。
- 所有设置标签都新增效果预览，预览图会叠加小型示例 UI，便于直观看到界面底色、文字增强和背景滤镜效果。
- 新增轻量原生 WebGL 液态玻璃增强层，不支持时自动降级为 CSS 玻璃效果。

### 优化

- 下拉框改为自定义可读浮层，避免原生 select 在透明背景下文字和背景混在一起。
- 设置页标签栏切换时固定尺寸，未选中标签保持纯文字透明状态。
- 设置页上传区、图库、按钮、滑块、状态块和高级 CSS 区域统一为更克制的透明玻璃风格。
- Dashboard 原生选项卡增强可读性，选中项保留清晰下划线，未选中项减少视觉噪声。
- 顶栏底部分隔线在透明主题下会被移除，避免顶栏与内容之间出现细黑线。
- 移动端布局继续保持单列适配，减少按钮、预览和长文本溢出风险。

### 修复

- 修复设置页下拉菜单被容器裁剪，导致选项被挡住的问题。
- 修复插件设置页中选项卡玻璃框残留在原位置的问题。
- 修复禁用美化但仍存在背景图时，预览示例 UI 残留显示的问题。
- 修复深色背景下保存按钮文字可读性不足的问题。
- 修复部分玻璃材质产生黑色像素块、点击大范围抖动和多余 hover 光效的问题。

### 兼容性

- 不新增后端 API，不新增配置字段。
- 保留背景图库、随机背景、主题色联动、上传格式识别和删除二次确认逻辑。
- 继续不修改 AstrBot 源码，仅通过插件设置页和运行时 Dashboard 注入样式实现。

## 0.3.0

### 新增

- 新增背景图库，支持上传多张 WebUI 背景图。
- 新增缩略图库，可在设置页点击缩略图一键切换当前背景。
- 新增“打开或刷新 WebUI 时随机背景”开关，开启后每次进入 Dashboard 会从图库随机选择背景。
- 新增“拉伸铺满”背景填充方式，可在不裁切图片内容的情况下铺满窗口。
- 新增 README 效果展示截图，方便在插件详情页和仓库首页查看实际效果。

### 优化

- 上传图片现在会以真实文件头识别格式，兼容“文件名是 jpg、实际内容是 png”的图片。
- 上传图片只加入图库，不再自动替换当前背景，避免误切换正在使用的壁纸。
- 当前背景切换、随机背景和删除当前背景后，会自动重新读取壁纸主题色。
- 缩略图预览使用插件 API 读取，避免登录态或浏览器缓存导致预览失效。
- 稳定背景视口尺寸，减少页面出现滚动条时背景横向抖动。

### 修复

- 修复插件设置页在 sandbox iframe 中调用浏览器 `confirm()` 被拦截，导致删除背景无反应的问题。
- 修复单张图库开启随机背景时重复写入配置、反复重新计算主题色的问题。
- 修复部分上传图片因扩展名和真实内容不一致被拒绝的问题。

### 兼容性

- 最低 AstrBot 版本要求：`>=4.26.0-beta1`。
- 继续不修改 AstrBot 源码，仅通过插件 Web API、插件设置页和运行时 Dashboard 注入实现。
- 旧版单背景配置会自动迁移到 `background_images` 图库列表。
